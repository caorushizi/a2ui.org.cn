# A2UI 编辑器 (Composer)

使用 **CopilotKit A2UI Widget Builder** 交互式地构建 A2UI 小部件。

[![A2UI Composer](assets/A2UI-widget-builder.png)](https://go.copilotkit.ai/A2UI-widget-builder)

**[启动小部件构建器 →](https://go.copilotkit.ai/A2UI-widget-builder)**

## 功能介绍

- 可视化地实验 A2UI 组件
- 通过描述你想要的内容生成 A2UI JSON
- 查看实时预览
- 复制 JSON 以在你的代理中使用

由 [CopilotKit](https://www.copilotkit.ai/) 团队构建。
